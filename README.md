# Tribute Page - Internship Task

**Domain:** Web Development  
**Task No:** 2  
**Submitted to:** Oasis Infobyte  

---

## Objective
To design and develop a **Tribute Page** using only **HTML** and **CSS**, focusing on layout structure, typography, and balanced design.  
The objective was to create a visually appealing page that pays tribute to a notable personality while showcasing strong understanding of front-end design principles.

---

## Project Overview
This project **Tribute Page** was built purely with **HTML and CSS**, emphasizing the **art of simplicity and visual balance**.

While it appears simple, this project provided an excellent opportunity to:

- Explore visual hierarchy, spacing, and alignment  
- Practice clean, semantic HTML structure  
- Strengthen understanding of CSS styling and layout techniques  
- Develop an eye for minimal yet impactful UI design  

---

## Tools & Technologies Used
- **HTML5:** For semantic and structured markup  
- **CSS3:** For styling, layout, and typography  
- **Visual Studio Code:** Development environment  
- **Git & GitHub:** For version control and project hosting  

---

## Steps Performed
1. Planned and outlined the tribute content.  
2. Structured the page using semantic HTML tags like `<header>`, `<section>`, and `<footer>`.  
3. Applied CSS for typography, spacing, and color balance.  
4. Ensured proper alignment and responsiveness across screen sizes.  
5. Reviewed and refined visual presentation for consistency.  

---

## Outcome
- A clean and aesthetically pleasing **Tribute Page** with balanced layout and consistent styling.  
- Enhanced understanding of **CSS design principles** and **frontend aesthetics**.  
- A deeper appreciation for how design choices influence user experience.
